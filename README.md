# 📚 Fundamentos e Exercícios em Java (CLI)

## 💻 Sobre o Projeto

Este repositório funciona como um laboratório prático, reunindo uma série de atividades, exemplos e scripts desenvolvidos durante meus estudos de **Lógica de Programação e Java**. O foco principal deste projeto é a resolução de problemas, a criação de algoritmos eficientes e a consolidação da base estrutural da linguagem, habilidades que considero fundamentais para o raciocínio analítico, *troubleshooting* e suporte em TI.

## 🎯 O que eu aprendi

* **Lógica de Programação:** Construção de algoritmos passo a passo para resolver problemas específicos.
* **Estruturas de Controle:** Domínio prático sobre laços de repetição (`for`, `while`) e estruturas condicionais (`if/else`, `switch`).
* **Manipulação de Dados:** Uso de variáveis, tipos primitivos, entrada e saída de dados via terminal (CLI).
* **Depuração de Código:** Identificação e correção de erros lógicos durante o desenvolvimento dos scripts.

## ✨ Funcionalidades

O repositório contém diversos scripts independentes, abordando diferentes conceitos:
* 🟢 **Atividades 1 a 9:** Resolução de desafios lógicos, matemáticos e de formatação de dados.
* 🟢 **Exemplos Práticos:** Arquivos de testes rápidos para validar a sintaxe e o comportamento de funções nativas do Java.

## 📁 Arquitetura do Projeto

Por se tratar de um repositório de estudos e testes rápidos, a estrutura é focada na simplicidade, com os arquivos mantidos na raiz ou separados por tópicos de estudo.
